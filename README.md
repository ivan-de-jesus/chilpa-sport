# Chilpa Sport

App móvil sencilla para controlar partidos de fútbol 7.

## Funciones

- 4 cuartos de 12 minutos.
- Cronómetro descendente.
- Marcador.
- Registro de goles por número de jugador.
- Registro de faltas.
- Tarjetas amarillas y rojas a pantalla completa.
- Deshacer último evento.
- Victoria normal: 3 puntos al ganador y 0 al perdedor.
- Victoria por shootout: 2 puntos al ganador y 1 al perdedor.
- Vista de árbitro protegida con contraseña.
- Sesión del árbitro guardada en localStorage.
- Recuperación del partido guardado.
- Vista pública para espectadores.

## Contraseña del árbitro

CHS

## Uso local

Abre `index.html` en el navegador.

## Publicación

Puedes subir estos tres archivos a GitHub Pages, Netlify, Vercel o Cloudflare Pages.

## Importante

Esta primera versión usa localStorage. Por eso, la vista pública solo verá el partido en el mismo navegador o dispositivo. Para que espectadores en otros celulares vean el partido en tiempo real, después habrá que conectar la app a Firebase, Supabase o una API propia.
